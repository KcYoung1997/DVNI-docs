Packet Sniffing¶

class scenarios.PacketSniffing.Import(teacher=False, developer=False, seed=None)[source]¶

Bases: scenarios.Scenario

This scenario requires the user to use wireshark/tcpdump to capture FTP credentials being sent on the network.

Parameters:
  • teacher (bool) – Is the user a teacher.
  • developer (bool) – Is the user a developer.
  • seed (str) – str: Random number generator seed, likely student ID.
