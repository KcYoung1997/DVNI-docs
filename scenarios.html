Scenarios¶

class scenarios.Scenario(teacher=False, developer=False, seed=None)[source]¶

Bases: object

Base scenario class. Performs the functions to run and document a scenario. Should be extended by any other implemented scenarios, all public methods are extensible. Extension classes should have the class name Import as per the start file.

Parameters:
  • teacher (bool) – Is the user a teacher.
  • developer (bool) – Is the user a developer.
  • seed (str) – str: Random number generator seed, likely student ID.
